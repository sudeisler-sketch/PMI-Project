# PMI-Project

## Project Title
Analyzing Philip Morris International’s Stock Performance with Market and Macroeconomic Indicators

## Project Description
This project analyzes the long-term stock performance of Philip Morris International (PMI) and examines its relationship with broader market trends. For this milestone, the project includes data collection, exploratory data analysis (EDA), and hypothesis testing.

## Files in This Repository
- `DSA210_Project_Proposal.pdf`: project proposal
- `data_collection.py`: downloads PMI and S&P 500 monthly data and creates the merged dataset
- `pmi_project_data.csv`: cleaned and merged dataset used in the analysis
- `eda.py`: exploratory data analysis and visualizations
- `hypothesis_tests.py`: statistical hypothesis tests

## Data Sources
- Yahoo Finance: PMI stock data
- Yahoo Finance: S&P 500 benchmark data

## Milestone 1 Content
This milestone includes:
- Data collection and preparation
- Descriptive statistics and visual analysis
- Correlation analysis
- Hypothesis testing on monthly returns

## Main Preliminary Findings
- PMI weekly returns are significantly different from zero at the 5% significance level.
- PMI weekly returns and S&P 500 monthly returns show a statistically significant positive relationship.

## How to Run
1. Install the required packages listed in `requirements.txt`
2. Run `data_collection.py`
3. Run `eda.py`
4. Run `hypothesis_tests.py`
